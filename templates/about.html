About WhoLoL.gg

WhoLoL.gg is a clone of websites like op.gg created using Flask (a micro web framework written in Python).

This site is a trimmed down, faster alternative to sites like op.gg. This base version of the site allows a user to search a League of Legends summoner name. If the searched player is currently in a match, the site will return information about said match. The information is very trimmed down, displaying just the champion names, the summoner names, and ranked data for all players in the match.

The site is very modular, and more information about the current match could be added. Some examples of additional content: a player's champion mastery of the champion they are playing as, the icon for their current ranking, and their summoner level. If it's available through Riot's API data for a live game, it can be added.
